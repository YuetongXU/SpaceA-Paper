
# 项目说明文档
本项目包含一系列生物信息学分析相关的脚本，主要用于处理空间三维基因组数据，涵盖数据预处理、数据过滤、Higashi分析及图表绘制等核心流程。

## 目录结构与功能说明
### 1. Preprocessing
该文件夹包含数据预处理全流程的相关脚本，核心功能如下：
- 数据质控：对原始数据进行质量控制；
- 序列比对：完成测序数据的比对分析，匹配参考基因组；
- 数据转换：将预处理后的数据转化为Cluster互作数据格式，为后续互作分析提供基础。

### 2. Filter_SpotsClustersReads
该文件夹聚焦于数据过滤相关脚本，核心处理对象包括背景Spot过滤，Clusters过滤与Reads过滤。

### 3. Higashi
该文件夹围绕Higashi分析流程开发相关脚本，核心功能包括：
- 将数据转换为Higashi分析所需格式；
- 执行FastHigashi算法完成数据聚类分析；
- 对多个样本进行整合。

### 4. figs
该文件夹包含研究文章中所有主图、附图对应的分析与绘图脚本，核心分析方向：
- 细胞周期分析；
- 互作距离分析vs互作频率；
- 覆盖度分析；
- 其他图表。

## 通用说明
1. 各文件夹下的脚本均为研究流程的核心组件，建议按照「数据预处理→数据过滤→Higashi分析→图表绘制」的顺序执行；
2. 脚本运行前需确保依赖的生物信息学工具（如比对工具、聚类工具等）已安装配置；
3. 若需调整分析参数，可在对应脚本的参数配置段进行修改。
