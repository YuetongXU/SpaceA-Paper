# 项目说明文档
本项目包含一系列生物信息学分析相关的脚本，主要用于处理空间三维基因组数据，涵盖数据预处理、数据过滤、Higashi分析及图表绘制等核心流程。

## 目录结构与功能说明
### 1. Preprocessing
该文件夹包含数据预处理全流程的相关脚本，核心功能如下：
- 数据质控：对原始数据进行质量控制；
- 序列比对：完成测序数据的比对分析，匹配参考基因组；
- 数据转换：将预处理后的数据转化为Cluster互作数据格式，为后续互作分析提供基础。

### 2. Filter_SpotsClustersReads
该文件夹聚焦于数据过滤相关脚本，核心处理对象包括背景Spot过滤，Clusters过滤与Reads过滤。

### 3. Higashi
该文件夹围绕Higashi分析流程开发相关脚本，核心功能包括：
- 将数据转换为Higashi分析所需格式；
- 执行FastHigashi算法完成数据聚类分析；
- 对多个样本进行整合。

### 4. Figs
该文件夹包含研究文章中所有主图、附图对应的分析与绘图脚本，核心分析方向：
1. Contact_Distance
聚焦染色质互作距离相关的分析与可视化，探究不同样本 / 区域中染色质互作的距离分布特征、SpaceA 分析中单 spot 的覆盖度等。
2. Cellcycle
围绕细胞周期展开分析，探究不同细胞周期阶段的染色质互作特征、基因表达模式等。
3. Contact_Num
分析染色质互作数量相关特征，统计不同样本 / 细胞类型 / 区域的染色质互作总数、有效互作数目等，并可视化展示。
4. Heatmap_SpaceA_BulkHiC
生成SpaceA 分析结果与 Bulk Hi-C 数据对比的热图，直观展示两种技术在染色质互作检测上的一致性 / 差异性。
5. Lineplot_E1value
通过折线图可视化 E1 值（推测为特定分子 / 表达 / 互作相关量化指标）的变化趋势，探究 E1 值在不同样本、发育阶段、聚类中的动态变化。
6. Contact_Heatmap
生成染色质互作热图（区别于与 Bulk HiC 对比的热图），聚焦 SpaceA 数据自身的染色质互作模式，包括特定区域、特定样本、特定聚类的互作热图。
7. Contact_Distance_Sample
在Contact_Distance基础上，进一步拆分到单个样本维度的染色质互作距离分析，聚焦样本间的互作距离差异。
8. NoSwitchRange_Expression
分析无开关区域（NoSwitchRange） 的基因表达特征，探究该区域的表达模式与染色质互作的关联。
9. Filter_Spot_Sprite
可视化spaSPRITE 技术中 Spot 过滤的过程与结果，评估过滤策略对数据质量的提升效果。
10. Pseudotime_Analysis
基于拟时序（Pseudotime）分析 可视化细胞 / Spot 的发育轨迹，探究染色质互作 / 基因表达随拟时序的动态变化。
11. UMAP_EmbryoSpatial
通过UMAP 降维可视化胚胎空间维度的细胞 / Spot 聚类特征，结合空间位置信息展示染色质互作 / 基因表达的空间分布模式。

## 通用说明
1. 各文件夹下的脚本均为研究流程的核心组件，建议按照「数据预处理→数据过滤→Higashi分析→图表绘制」的顺序执行；
2. 脚本运行前需确保依赖的生物信息学工具（如比对工具、聚类工具等）已安装配置；
3. 若需调整分析参数，可在对应脚本的参数配置段进行修改。
